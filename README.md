# home-jiuyou-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库本身不面向任何特定网站或域名，仅作为 HTML 文件的存储与展示空间。

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面，内容涵盖笔记、工具页面、演示页面等。  
每个页面均为自包含的 HTML 文件，方便直接通过 GitHub Pages 或其他静态托管服务访问。

## 目录说明

```
├── index.html          # 仓库首页 / 导航页（如有）
├── pages/              # 存放各类 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `pages/`：主要存放独立的 HTML 页面文件，按需组织子目录。
- `assets/`：存放多个页面可能共用的样式、脚本或媒体文件。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，可直接在浏览器中打开。
- 文件命名建议使用有意义的英文或拼音，便于识别。
- 页面内不应包含指向具体第三方网站的硬编码链接或营销内容。
- 归档的页面内容可能来自个人笔记、实验性项目或临时工具，不保证长期更新或兼容性。
